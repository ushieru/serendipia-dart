# Serendipia

![Serendipia]('./resources/serendipia.png')

Serendipia is a RESTful (Representational State Transfer) for the purpose of discovery, load balancing and fault tolerance.

## Documentation

```bash
# Use a default port
$ serendipia

# You can assign a port
$ serendipia --port 5000

# Or with alias
$ serendipia -p 5000
```
